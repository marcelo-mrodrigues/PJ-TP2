Retorna um dicionário de contexto para renderizar templates HTML base de gerenciamento.

Esta função auxilia na construção de páginas de gerenciamento que incluem um formulário (para adicionar/editar), uma lista de itens existentes e mensagens.

Parameters
requestO objeto HttpRequest do Django.
titleO título da página (ex: "Gerenciar Lojas").
form_objUma instância de Django Form (opcional), usada para renderizar o formulário.
existing_items_htmlUma string HTML com a lista de itens existentes.
additional_info_htmlUma string HTML com informações adicionais a serem exibidas.
Returns
Um dicionário contendo o contexto necessário para o template.

◆ process_loja_form()

core.utils.process_loja_form (   request)

Processa o envio de um formulário de Loja (adição ou edição).

Esta função encapsula a lógica de validação e salvamento de formulários de loja, retornando o formulário, um indicador de redirecionamento e mensagens adicionais.

Parameters
requestO objeto HttpRequest do Django, contendo os dados POST.
Returns
Uma tupla contendo:
  • form (LojaForm): A instância do formulário processado.
  • redirect_needed (bool): True se o formulário foi salvo com sucesso e um redirecionamento é necessário.
  • additional_info (str): Uma string HTML com informações adicionais ou mensagens de erro.

◆ search_products()

core.utils.search_products (   query = "")

Busca produtos com base em um termo de consulta e anota o menor preço para cada um.

A busca é realizada nos campos de nome, descrição, nome da categoria e nome da marca.

Parameters
queryO termo de busca (string). Se vazio, retorna todos os produtos.
Returns
Uma lista de dicionários, onde cada dicionário representa um produto com suas informações básicas e o menor preço encontrado em suas ofertas.
