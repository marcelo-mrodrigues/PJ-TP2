Detailed Description

Define os modelos de dados para o aplicativo 'core' do FoodMart.

Este arquivo contém as definições de todos os modelos Django que representam as entidades do sistema, como Categorias, Marcas, Produtos, Usuários, Lojas, Ofertas, Itens Comprados, Listas de Compras, Itens de Lista e Comentários. Também inclui um modelo para Produtos Indicados por usuários.

Cada modelo herda de django.db.models.Model e define campos com tipos de dados apropriados, validações, relacionamentos e metadados para administração.
