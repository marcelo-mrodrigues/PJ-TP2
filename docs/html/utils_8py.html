Detailed Description

Contém funções utilitárias para manipulação de dados e lógica de negócio do aplicativo 'core'.

Este arquivo agrupa funções auxiliares que interagem com os modelos do Django para buscar informações de produtos, gerenciar o carrinho de compras na sessão, realizar buscas no catálogo e renderizar componentes HTML para o backend.
